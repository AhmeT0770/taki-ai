# AI Studio uygulamanızı çalıştırın ve dağıtın

Bu depo, uygulamanızı yerelde çalıştırmanız için gereken her şeyi içerir.

Uygulamanızı AI Studio’da görüntüleyin: https://ai.studio/apps/drive/1GWn2AcigpZaNAuNJ5MKrpFibVfC_7dcR

## Yerelde Çalıştırma

**Ön koşul:** Node.js

1. Bağımlılıkları yükleyin:  
   `npm install`
2. Gemini API anahtarınızı [.env.local](.env.local) dosyasındaki `GEMINI_API_KEY` değişkenine ekleyin
3. Uygulamayı başlatın:  
   `npm run dev`
